## 初級

画面操作の話で出ますが、CUI（文字ベース）や API との対比がないと「何が嬉しいのか」が薄まりやすいです。

## 中級

ウィジェット、イベント、状態管理、ユーザビリティやアクセシビリティが設計の中心です。業務アプリでは権限に応じた画面制御もセットになります。

## 上級

ネイティブと Web のハイブリッド、レンダリングパイプライン、入力遅延とフレーム予算、デザインシステムとの整合が深掘りされます。

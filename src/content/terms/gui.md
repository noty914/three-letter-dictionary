## 初級

画面のアイコンやボタンで操作するユーザー向けの見た目・操作まわり。

メニューやウィンドウで直感的に触れるインターフェースの総称です。

## 中級

CUI（文字だけの画面）や API との対比がないと「何が嬉しいのか」が薄まりやすいです。ウィジェット、イベント、状態管理、ユーザビリティやアクセシビリティが設計の中心で、業務アプリでは権限に応じた画面制御もセットになります。

典型シーンは「非エンジニアが日常業務で使うアプリ」です。CLI より学習コストが低い一方、操作手順が増えると逆に遅くなるため、画面設計と自動化のバランスが重要です。

## 上級

ネイティブと Web のハイブリッド、レンダリングパイプライン、入力遅延とフレーム予算、デザインシステムとの整合が深掘りされます。
